# Window Manager

Windows 11 向けウィンドウ配置管理ツール。
グローバルホットキーでウィンドウを素早くタイル・整列・プリセット配置できます。

---

## 機能

| 機能 | 説明 |
|------|------|
| ポップアップメニュー | ホットキー（デフォルト `Ctrl+Alt+W`）でプリセット一覧を表示 |
| 即実行ホットキー | プリセットごとに個別ホットキーを登録してワンキー実行 |
| プリセット管理 | ウィンドウの配置（位置・サイズ）を名前付きで保存・編集・削除 |
| クイックレイアウト | 左半分・右半分・1/3分割など14種のレイアウトテンプレート |
| 現在配置の取り込み | 今開いているウィンドウの位置をそのままプリセット化 |

---

## 動作環境

- Windows 11（win32 API を使用）
- Python 3.8 以上（Windows 側）

> **注意：** WSL2 から実行する場合は `py.exe -3 main.py` を使用してください。

## インストール

```powershell
# 依存ライブラリのインストール
pip install pywin32 keyboard psutil
```

## 起動

```powershell
# コマンドプロンプト / PowerShell
py -3 main.py

# またはダブルクリック
start.bat
```

> **管理者権限について：** `keyboard` ライブラリはグローバルホットキーの検知に管理者権限が必要な場合があります。
> 動作しない場合は「管理者として実行」してください。

## 使い方

### 1. ポップアップメニュー

`Ctrl+Alt+W` を押すと画面中央にポップアップが表示されます。

- クリック または 数字キー（1〜9）でプリセットを実行
- `ESC` で閉じる
- `⚙ 管理メニュー` ボタンで管理画面を開く

### 2. 即実行ホットキー

管理メニューで各プリセットにホットキーを設定すると、ポップアップを開かずに直接実行できます。

```
例: Ctrl+Alt+1 → 横並び2分割を即実行
```

### 3. 管理メニュー

ポップアップの `⚙ 管理メニュー` から開きます。

#### プリセット登録

1. `＋ 新規` をクリック
2. 名前・ホットキーを入力
3. `＋ ウィンドウ追加` で対象ウィンドウを選択
4. クイックレイアウトドロップダウンで配置を選択
5. `保存` をクリック

#### 現在の配置を取り込む

1. 配置したいウィンドウを手動で並べる
2. 管理メニューで `現在の配置を取り込む` をクリック
3. 名前・ホットキーを入力して `保存`

---

## レイアウト値について

配置は画面作業領域（タスクバー除く）に対する比率（0.0〜1.0）で指定します。

| 値 | 意味 |
|----|------|
| x=0.0, w=0.5 | 左半分 |
| x=0.5, w=0.5 | 右半分 |
| x=0.0, w=1.0 | 横全体 |
| y=0.0, h=0.5 | 上半分 |

## ファイル構成

```
window-manager/
├── main.py            # エントリポイント、ホットキーデーモン
├── config_mgr.py      # JSON設定管理
├── window_mgr.py      # Win32 ウィンドウ操作
├── ui_popup.py        # ポップアップ選択メニュー
├── ui_management.py   # プリセット管理UI
├── config.json        # 設定ファイル（自動生成）
├── requirements.txt   # 依存ライブラリ
├── start.bat          # Windows 起動スクリプト
└── tests/
    ├── test_config_mgr.py
    └── test_window_mgr.py
```

## config.json の形式

```json
{
  "popup_hotkey": "ctrl+alt+w",
  "presets": [
    {
      "id": "uuid",
      "name": "横並び2分割",
      "hotkey": "ctrl+alt+1",
      "windows": [
        {
          "app": "chrome",
          "layout": { "x": 0.0, "y": 0.0, "w": 0.5, "h": 1.0 }
        },
        {
          "app": "code",
          "layout": { "x": 0.5, "y": 0.0, "w": 0.5, "h": 1.0 }
        }
      ]
    }
  ]
}
```
